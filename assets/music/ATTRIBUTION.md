**Before monetizing the channel**, open each track's Pixabay page and:
1. Confirm it is still covered by the Pixabay Content License (free for
   commercial use, no attribution required — but terms evolve).
2. Save a screenshot/PDF of the license line as proof against future
   Content ID disputes.
3. If any track has moved to a paid-only license, replace it and update
   this table.

Replace `MUSIC_TRACK` in `.env` to pin one track, or leave empty to rotate.
